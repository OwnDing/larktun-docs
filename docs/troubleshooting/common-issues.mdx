# 常见问题

本页适合维护一个持续增长的问题列表，优先覆盖最容易阻塞用户的场景。

## 服务无法访问

优先确认目标服务在本地或内网中是否本来就可达，再检查入口绑定和访问策略是否生效。

## 域名访问异常

确认域名解析是否生效、证书是否正确、入口地址是否与当前发布配置一致。

## 授权用户仍被拒绝

重点检查授权规则是否命中、访问凭据是否过期、访问来源是否满足限制条件。

## 链路偶发超时

从健康检查、上游服务负载、网络抖动和连接池限制几个方向逐一定位，并结合日志确认超时发生在哪一段链路。
