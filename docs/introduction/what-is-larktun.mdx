# 网站介绍

Larktun 可以作为 `larktun.com` 的产品文档入口，帮助用户快速理解“它是什么、适合解决什么问题、从哪里开始使用”。

## 适合解决的问题

- 需要让本地或内网服务被外部安全访问
- 需要为临时环境或分支环境提供稳定访问入口
- 需要在不扩大暴露面的前提下支持远程联调或问题处理

## 核心能力

- 用统一入口管理服务暴露、权限控制与访问记录
- 让团队成员在不同环境之间快速建立可验证的连接链路
- 为后续博客、案例和最佳实践提供一致的概念基础

## 典型使用场景

- 产品与测试团队远程访问预发布环境
- 运维团队按最小暴露原则开放内网系统
- 客户支持团队通过临时授权完成问题诊断
