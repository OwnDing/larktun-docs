# Larktun Documentation Hub

Larktun is a secure networking and remote access platform for distributed teams. This documentation site brings together product overviews, deployment guidance, onboarding steps, troubleshooting references, and scenario content in one place.

## Recommended reading path

1. Start with product overview and core concepts to understand Headscale, zero-trust networking, policy templates, and audit behavior.
2. Continue into getting started to open an account, onboard the first devices, and complete an initial verification flow.
3. If your rollout needs a more formal design, continue to deployment modes and configuration.
4. If something fails, jump to troubleshooting for device, policy, path, and audit diagnostics.
5. Use the blog and showcase to go deeper into rollout notes and real-world examples.
