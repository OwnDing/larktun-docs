The first branded version of the Larktun documentation site is now in place, with docs, blog, and showcase entry points plus bilingual support for Chinese and English.

{/* truncate */}

This update solves three things:

- It syncs official product copy, brand assets, and homepage screenshots into the help center.
- It reshapes the docs around zero-trust networking, remote control, security audit, and deployment modes.
- It adds a GitHub Pages workflow so the site is ready to publish after the repository is pushed.

The next step is to add deeper control-plane screenshots, real field-level explanations, CLI or API examples, and more detailed enterprise delivery guides.
