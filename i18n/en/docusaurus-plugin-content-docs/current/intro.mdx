# Larktun Documentation Hub

Larktun is a SaaS zero-trust networking and remote access platform for teams. This documentation focuses on four priorities:

- SaaS and multi-tenancy: each tenant runs as an isolated network
- Independent ACL per tenant: policy is scoped and enforced per tenant
- Custom relay support: shared relay, dedicated relay, or user-managed relay
- Lightweight and stable runtime: low memory usage, non-intrusive behavior, reliable connectivity
