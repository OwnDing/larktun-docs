# 常见问题

本页覆盖初期最常见的阻塞：ACL 拒绝、自建中继不可用、客户端占用异常和链路不稳定。

## 访问被拒绝（ACL）

先确认访问请求是否发生在正确租户内，再检查 ACL 规则是否命中、授权是否过期、审批是否完成。

## 自建中继不可用

检查中继地址与端口是否正确、节点健康状态是否正常、是否存在网络防火墙或证书问题。

## 客户端占用偏高

确认是否开启低内存配置、是否存在异常重试、是否有过量日志级别导致资源波动。

## 访问偶发断连

优先检查当前网络质量和中继路径，再查看自动重连是否生效，以及目标端是否负载过高。

## 用户反馈“被打扰”

检查客户端运行模式是否为后台静默，确认是否存在不必要的弹窗、通知或前台抢占行为。
