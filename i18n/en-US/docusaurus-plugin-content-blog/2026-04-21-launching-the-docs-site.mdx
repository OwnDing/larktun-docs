The first version of the Larktun documentation site scaffold is now in place, with dedicated entry points for docs, blog content, and showcase pages, plus a bilingual structure for Chinese and English.

{/* truncate */}

This initial scaffold mainly solves three things:

- It provides a unified navigation and homepage structure for `larktun.com` help content.
- It makes the overview, getting started, and troubleshooting sections ready for real product content.
- It reserves long-term space for release posts, implementation notes, and customer stories.

The next step is to replace the placeholder copy with real screenshots, CLI examples, configuration details, and case studies.
