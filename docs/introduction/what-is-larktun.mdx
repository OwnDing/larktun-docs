# 产品介绍

Larktun 云雀通是一个 SaaS 化零信任网络平台，用于帮助团队安全连接设备与内部服务。它把网络接入、访问控制、审计与远程访问体验放在同一控制平面中，适合研发、运维和企业远程办公场景。

## 产品定位

- 面向多租户团队的远程网络与访问治理平台
- 用一个控制台统一管理设备、用户、策略和审计信息
- 提供稳定、低打扰、可扩展的跨网络访问体验

## 核心能力

### 1. SaaS 化与多租户隔离

- 平台以 SaaS 方式提供控制能力，开通快、维护成本低
- 每个租户拥有独立网络边界，租户之间默认隔离
- 租户间的设备目录、访问记录和安全事件互不混用

### 2. 每租户独立 ACL

- ACL 以租户为边界进行配置与生效
- 可以按用户组、设备标签、访问时间窗进行细粒度授权
- 可结合审批、临时授权与自动回收减少权限滞留

### 3. 用户可自定义中继服务器

- 支持共享中继快速上线
- 支持专用中继提升可控性
- 支持用户自建中继服务器，用于满足区域、性能或合规要求

### 4. 低内存、不打扰、稳定性

- 客户端常驻时内存占用低，适合长期后台运行
- 默认后台工作，不影响用户前台操作
- 在复杂 NAT 或跨地域网络下具备稳定连接与自动恢复能力

## 典型使用场景

- 远程研发团队安全访问代码仓库、构建节点与调试环境
- IT 运维与 MSP 统一管理多客户设备并保持权限隔离
- 企业远程办公为内网系统提供可审计、可回收授权的访问入口
