# 常见问题

本页优先覆盖设备接入、策略拒绝、链路不稳定和审计缺失等最常见的阻塞场景。

## 服务无法访问

先确认目标设备是否在线，再检查访问策略、中继状态与目标端本身是否可达。

## 域名访问异常

如果你为内部系统配置了 Web 入口，需要确认入口地址、证书和当前工作空间下的发布配置是否一致。

## 授权用户仍被拒绝

重点检查策略模板是否命中、授权是否过期、是否需要审批，以及企业 SSO 或 MFA 是否已完成。

## 链路偶发超时

从中继质量、跨地域网络抖动、目标端负载和会话链路几个方向逐一定位，并结合日志确认问题出现在哪一段路径。
