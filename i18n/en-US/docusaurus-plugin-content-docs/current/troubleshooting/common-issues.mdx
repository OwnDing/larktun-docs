# Common Issues

This page should grow into a high-frequency issue list, focused on the problems most likely to block users.

## Service is unreachable

First confirm the target service is reachable inside the local or internal network before checking entry point binding and access policy behavior.

## Domain access problems

Verify DNS propagation, certificate validity, and whether the entry point matches the published configuration.

## Authorized users are still blocked

Check whether the authorization rule was matched, whether the credential expired, and whether the client satisfies source restrictions.

## Intermittent timeouts

Investigate health checks, upstream load, network jitter, and connection pooling. Use logs to identify where in the path the timeout occurs.
