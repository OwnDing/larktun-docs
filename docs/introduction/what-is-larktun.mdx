# 产品介绍

Larktun 云雀通是一个将零信任网络、稳定远程控制与集中运维结合在一起的平台，面向分布式团队提供安全远程办公、设备接入与访问治理能力。

![Larktun 产品首页预览](/img/larktun-homepage.png)

## 产品定位

- 为分布式团队打造安全网络与远程控制平台
- 用一个控制台统一管理设备、策略、访问日志和安全审计
- 在不依赖传统 VPN 的前提下，提供跨地域、跨平台的稳定互联能力

## 核心能力

- 零信任安全网络：基于 WireGuard 与 Headscale 构建加密隧道，并默认按最小信任原则接入设备
- 极速远程控制：针对 RDP、SSH、VNC 优化链路，支持远程桌面、数据运维和混合办公
- 安全与合规：支持 MFA、企业 SSO、策略模板、审批和审计日志，满足更正式的安全要求
- 灵活部署：支持共享中继、专用中继，以及私有或混合部署路径

## 典型使用场景

- 远程研发团队安全访问代码仓库、构建节点和测试资源
- IT 运维与 MSP 统一管理多客户服务器、桌面资产和健康状态
- 企业远程办公场景下，为内部系统提供带审批和审计的安全入口

## 官方亮点

- 免费版可快速试用，适合先完成小规模验证
- 从注册、接入设备到验证访问链路，可以在较短时间内完成首轮部署
- 商务与企业方案支持更高 SLA、专用支持和更灵活的部署方式
