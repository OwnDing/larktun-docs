# Product Overview

Larktun combines zero-trust networking, low-latency remote access, and centralized operations into a single platform for distributed teams.

![Larktun homepage preview](/img/larktun-homepage.png)

## Positioning

- A secure networking and remote access platform for distributed teams
- One control plane for devices, policies, session logs, and security audit trails
- Stable connectivity across regions and platforms without relying on a legacy VPN model

## Core capabilities

- Zero-trust networking built on top of WireGuard and Headscale
- Low-latency remote control paths for RDP, SSH, and VNC
- Identity and compliance features such as MFA, enterprise SSO, approvals, and audit logs
- Flexible rollout choices including shared relay, dedicated relay, and private or hybrid deployment

## Common scenarios

- Secure access to code repositories, build nodes, and debugging environments for distributed engineering teams
- Centralized management of customer servers and desktop assets for IT operations and MSP teams
- Audited, approval-based remote access to internal systems for enterprise remote work
