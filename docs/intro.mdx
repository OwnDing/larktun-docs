# Larktun 文档中心

Larktun 云雀通是面向团队的 SaaS 化零信任网络与远程访问平台。本文档围绕以下四个重点展开：

- SaaS 化与多租户：每个租户都是隔离网络，资源、身份和日志彼此独立
- 独立 ACL：每个租户都可以定义独立访问控制策略
- 自定义中继：支持共享中继、专用中继和用户自建中继服务器
- 轻量稳定：客户端低内存占用、后台运行不打扰、跨网络访问稳定
