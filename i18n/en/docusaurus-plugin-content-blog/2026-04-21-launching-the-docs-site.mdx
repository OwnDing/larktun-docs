The first version of the Larktun documentation site scaffold is now in place with three entry points: docs, blog, and showcase, plus bilingual support for Chinese and English.

{/* truncate */}

This release is organized around four capability tracks:

- SaaS multi-tenancy with tenant-isolated networks
- Tenant-level ACL with independent policy control
- User-managed relay support alongside shared and dedicated modes
- Low-footprint and stable runtime with non-intrusive endpoint behavior

Next, we will expand operational examples, scenario playbooks, and troubleshooting depth so the site can be used directly in delivery workflows.
